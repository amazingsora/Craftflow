### 4. AI 生成 Prompt 架構調整

**進入生成的欄位（`✦` 標示）**

| 欄位 | 方式 |
|---|---|
| `ai_prompt` | **獨立編譯，置於 prompt 最前端**（最高 SD 注意力權重） |
| `core_traits` 外貌/個性 | 與角色描述一起編譯 |
| `age` 年齡 | 與角色描述一起編譯 |
| `color` 代表色 | 背景色用途 |
| `concept_images[0]` | vision model 萃取視覺特徵（30~60% 參照） |

**從生成移除的欄位**
- `voice_style`（說話風格）— 對圖像無實質影響，已從 `parts` 移除

**Final prompt 結構**
```
[ai_prompt 獨立編譯] + [角色描述編譯] + [人設表後綴 + 背景色標籤]
     ↑ 最高強制力          ↑ 角色特徵         ↑ layout/style 固定標籤
```

---

### 5. UI 視覺區分

AI 生成欄位在 label 前加 `✦`（accent 色）作為識別符號：

- `✦ 年齡`
- `✦ 外貌 / 個性特徵`
- `✦ AI 提示詞`（獨立編譯，置於 prompt 最前端，強制力最高）

其餘欄位（說話風格、行為模式、創作筆記、生日）無標示 → 純角色資料，不進入 AI 生成。
