## 本次完成（2026-05-11）

### 新增後端 API

| Endpoint | 說明 | 狀態 |
|----------|------|------|
| `POST /api/v1/art/lineart` | 草稿 → 線稿（ComfyUI ControlNet） | ✅ 完成，待實測 |
| `POST /api/v1/art/generate` | 文字 Prompt → 圖片（SDXL txt2img） | ✅ 完成，待實測 |
| `POST /api/v1/art/compose` | 草圖 + 問題 → 構圖意見 + 參考圖 | ✅ 完成，待模型下載 |

### 新增後端服務

- `backend/app/services/comfyui_client.py` — ComfyUI REST 包裝器（上傳/提交/輪詢/下載）
- `backend/app/services/ai/art_service.py` — 新增 `compose_ask()`（Ollama 視覺分析 → 意見 + SDXL prompt）
- `tools/Craftflow/diffusion/workflows/text_to_image.json` — SDXL txt2img workflow

### 新增前端（React + Vite）

- `frontend/` — 專案建立完成（package.json、vite.config.js、index.html）
- `frontend/src/App.jsx` — 三個 Tab + 底部歷史縮圖列（跨 Tab 保留）
- `frontend/src/components/ProcessTab.jsx` — 草稿 → 線稿
- `frontend/src/components/GenerateTab.jsx` — 文字 → 生圖（Prompt / Negative / Steps / Seed / 尺寸）
- `frontend/src/components/ComposeTab.jsx` — 草圖問答（Ollama 視覺 + ComfyUI 生圖）

### 修正

- `sketch_to_lineart.json` 移除 `ScribblePreprocessor` 節點（需 comfyui_controlnet_aux 才有）→ 直接將草圖餵給 ControlNet
- `comfyui_client.submit_workflow()` 現在正確抛出 ComfyUI 的 node 錯誤訊息
- `ollama_client.py` timeout 調高至文字 300s / 視覺 600s（for 32B 模型）
