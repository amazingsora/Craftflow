## 2. 草稿分析 Prompt 優化 (art_service.py)

### sketch_critique（草稿模式）
- 舊版：泛談構圖、解剖、線條
- 新版：**逐部位分析**（眼睛/五官、頭髮、手部、身體、衣物皺褶、輪廓線）
- 修正清單要求具體到部位（例：「右手食指第二關節偏短」而非「修正手指」）

### finished_critique（完成/半完成模式）
- 舊版：含解剖分析
- 新版：移除解剖，**聚焦配色、光影、構圖**三大面向
- 配色分析包含：主色調、冷暖色溫、強調色
- 光影分析包含：光源一致性、立體感、反光高光建議

## 3. Ollama 視覺分析加速 (ollama_client.py)

- 新增 `_resize_for_vision()`：使用 Pillow 將圖片縮至 **768px 長邊**再送 Ollama
  - 效果：大圖（如 A4 300dpi）從 3-5MB 縮為 ~100KB，分析時間顯著縮短
  - 視覺理解品質不受影響（視覺模型內部解析度固定）
- `analyze_image_bytes()` 新增 `options` 參數，傳遞 `num_predict`、`temperature`
- compose_ask 設定 `num_predict=350`、`temperature=0.4` 限制回覆長度
- 新增 Pillow 至 `requirements.txt`（需 `docker compose up -d --build` 重建）

## 4. 草圖問答生成圖優化 (art_generate.py / art_service.py)

### 問題根源
- ControlNet scribble 強度過高（0.7）→ 草稿空白處不補、不完整草稿造成肢體分離
- SDXL prompt 無風格限制 → 生成結果偏向完稿

### 最終決策
- compose endpoint 改回 **txt2img**（放棄 ControlNet scribble）
  - 理由：ControlNet 適合完整線稿，不適合創作中的不完整草圖
  - 構圖意圖改由 Ollama 生成的 prompt 保留
- 自動追加 suffix：`complete figure, all body parts visible, pencil sketch, rough sketch, monochrome, sketch style`
- 生成解析度依原圖比例自動計算，上限 1024px（無需手動設定）

## 6. 角色管理模組（全新）

### 架構：作品（Project）→ 角色（Character）兩層

#### Project 新增欄位
| 欄位 | 說明 |
|---|---|
| `genre` | 作品類型（玄幻/現代都市/科幻/古風/BL-GL/輕小說/其他） |
| `status` | 創作狀態（構思中/撰寫中/修稿中/完稿），附顏色 badge |

#### Character 新增欄位
| 欄位 | 說明 |
|---|---|
| `ai_summary` | AI 整理的角色設定檔（Markdown 格式） |
| `portrait_path` | 概念圖路徑（草稿/線稿/完稿皆可） |

#### SQLite 自動 Migration
- 新增 `_migrate()` / `_add_columns()` 於 `database.py`
- 啟動時自動 `ALTER TABLE ADD COLUMN`（若欄位不存在）
- 無需 Alembic，適合快速迭代

#### 新增 API 端點
| 端點 | 說明 |
|---|---|
| `GET /characters/default-project` | 取得或自動建立預設作品 |
| `POST /characters/{id}/summarize` | AI 整理角色筆記為設定檔 |
| `POST /characters/{id}/portrait` | 上傳概念圖 |
| `GET /characters/{id}/portrait` | 讀取概念圖 |

#### 前端 CharacterTab 頁面流程
```
作品列表（卡片：名稱、類型、狀態badge、描述、角色數）
  └─ [+ 新增作品] 或點入作品
       └─ 角色列表（麵包屑導覽）
            ├─ [編輯作品] inline 展開表單
            └─ [+ 新增角色] 或點入角色
                 └─ 角色詳細
                      ├─ 左上：AI 整理資訊 + [AI 重新整理]
                      ├─ 左下：概念圖上傳（拖曳/點擊）
                      ├─ 左底：[根據角色生成 AI 圖]
                      └─ 右欄：可編輯欄位（特徵/行為/說話風格/筆記）+ [儲存所有變更]
```
